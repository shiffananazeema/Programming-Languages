`````
Creating and Running a Project
`````````````````````````
Android Studio
````````````
  1. File -> New -> New Flutter Project -> Next -> Name 
  2. Select the Device -> Poco X2 -> Run 

Visual Studio Code
```````````````
  1. Ctrl + Shift + P -> Flutter: New Project -> Destination -> Name
  2. Ctrl + Shift + P -> Flutter: Select Device -> Poco X2
  3. Ctrl + ` -> Terminal -> flutter run -> r -> hot reload
